# Smart Gym Tracker CLI

A command-line workout tracker written in C++ that allows users to log, view, and delete strength workouts while saving workout history to a text file.

This project was built to practice object-oriented programming, file I/O, STL containers, and memory management using modern C++.

---

## Features

- Log strength workouts
- Save workouts to a text file
- Load previous workouts when the program starts
- View workout history
- Delete workouts by ID
- Persistent workout storage between sessions

---
